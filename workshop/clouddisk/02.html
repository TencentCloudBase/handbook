第一节：导入项目与体验

学习开源代码项目，能够让我们以最直接、最直观的方式来了解一个实际的完整项目到底是如何开发出来的，这就像学画画、书法时的临摹。通过对实际的代码项目一句句加以分析，你就能学到很多教程、文档里没有的知识组成方式，在知识点的应用上会给你一个全新的角度。在学习项目的源代码时，我们需要先把项目跑起来，体验一下，了解到底有哪些功能。

1、下载源代码的两种方式

通常我们都会把源代码放到开源代码共享平台Github上面，在Github上面你可以下载到世界上最优秀的程序员们开源出来的软件，它就像是少林寺的藏经阁一样，揽尽江湖之上的至高武学。

1.1 在Github上下载源代码压缩包

打开口袋网盘的Github地址，点击页面上里的绿色按钮Clone or download，然后再点击Download ZIP，这样Github的源代码压缩包就能下载下来了（建议大家用1.2里的方式）。

源代码地址：口袋网盘Github地址

 

1.2 使用Git命令下载源代码

首先我们在电脑上新建一个文件夹，比如文件名名称为tcb，然后打开终端命令行工具（Windows电脑为cmd命令提示符，Mac电脑为终端Terminal），先cd进该文件夹，方法如下：

Mac电脑 比如你把tcb文件夹放在Mac的下载downloads里面，我们需要输入以下命令之后按Enter：

1
cd downloads/tcb

Windows电脑 我们需要先找到tcb文件夹的路径，比如C:\download\tcb（可以选择文件夹之后，右键点击属性，在位置里即可看到文件夹的路径），需要输入命令按Enter:

1
cd /d C:\download\tcb

通常情况下cd命令即可切换路径，但是部分windows电脑直接使用cd命令无法切换，可以使用上面的cd  /d 命令，cd是change directory的缩写，这里 cd+空格+文件夹的路径，表示进入该文件夹。当然你也可以直接输入cd和一个空格之后把文件夹用鼠标拖到终端窗口。

然后再在命令行里输入如下git clone命令并按Enter执行，该命令会把Github上面的开源项目源代码下载到电脑里：

1
git clone https://github.com/TCBWorkshop/clouddisk.git

打开你之前建好的tcb文件夹，就可以在文件夹里看到使用Git下载好的clouddisk文件夹了。

2、导入项目并配置环境

2.1 导入项目到开发者工具

点击微信开发者工具的工具栏里的项目–导入项目来将clouddisk导入到开发者工具里。

  • 项目名称可以任意填写比如“口袋网盘小程序”，
  • 项目路径为tcb文件夹下的clouddisk；
  • AppID则需要填入我们自己的AppID（在准备时找到的AppID，如果之前填入过，输入框右侧有个绿色的下拉三角可以选择）。

当我们在开发者工具的模拟器里看到小程序的界面时，就说明小程序已经导入并编译成功啦。

在开发者工具的调试器里可能会有报错信息，这是因为我们还没有配置这个小程序所需要的环境。后面的步骤就是为了解决这个问题。

 

2.2 确认小程序的appid

使用开发者工具打开小程序根目录下的project.config.json文件，在文件里找到”appid”的配置，确认里面的值为你的小程序appid，如果不是，可以在这里直接修改（注意双引号””不要删了）。

也可以打开开发者工具的工具栏右上角的详情标签，在基本信息里也可以看到AppID的信息，如果该AppID不为你的小程序AppID，也可以在这里修改，这种方法是上面的方法是一样的。

 

2.3 基础库版本与其他配置

打开开发者工具的工具栏右上角的详情标签，在本地设置里将调试基础库的版本修改为最新版本（这一步对新开发的小程序来说不是必须的，但是对很多较旧的项目而言就很重要了）。勾选增强编译以及不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书。

增强编译可以让小程序支持很多ES6及以上的语法特性，不校验域名可以让我们不需要担心域名合法性的问题。之前有编程经验的童鞋可以深入了解一下，零基础的朋友可以以后来了解，现在直接勾选就可以啦。

2.5 添加user集合并修改权限

口袋网盘小程序会使用到数据库，打开云开发控制台，在数据库管理页新建一个user集合，然后将集合的权限设置为所有用户可读，仅创建者可读写。

3、体验小程序

3.1 在开发者工具上体验小程序

完成以上步骤之后，点击开发者工具栏的编译标签，就能执行小程序的代码点击开发者工具工具栏上的云开发标签，会弹出云开发控制台，在云开发控制台的user集合里数据是空的，而存储里也是没有相关文件的。我们待会再来看添加内容之后的变化。

 

3.2 在手机上体验小程序

点击开发者工具栏的预览标签，会将小程序的源代码上传到云端并生成小程序的体验二维码，用手机微信扫描二维码，就能在微信里打开这个体验版的小程序。在这个小程序里完成

  • 登录小程序；
  • 新建一个文件夹，并在该文件夹里上传3个以上的文件（会弹出微信聊天对话框），观察文件夹后的数字变化；
  • 长按上传的文件，会获取到文件的链接，使用手机浏览器打开链接试试；
  • 新建一个相册，并在该相册了上传一些照片，观察相册集封面的变化；

当我们修改代码之后，都需要重新点击预览标签，更新好的代码文件才会上传；而且生成的体验小程序的二维码只有该小程序的开发者绑定的微信才能扫码体验，其他人的微信是没有权限的哦。

 

3.3 观察云开发控制台

再次点击开发者工具的云开发标签，观察user集合内的数据，这些数据它们有哪些字段，这些字段大致代表了什么意思？以及云存储里的文件有哪些？点击云存储的文件详情，看看下载地址是啥，并把下载地址复制粘贴到浏览器看看什么效果？

当做完以上工作，我们相当于完整复现了这个小程序，以后是可以直接上传并发布出去并在实际中使用的，我们也可以边体验边了解背后的逻辑，后面章节我们也会教大家如何结合代码来理解小程序功能是怎么实现的。初学者即使不懂代码也能对背后的逻辑有一定的了解，对今后的深入学习也会有很大的帮助。
